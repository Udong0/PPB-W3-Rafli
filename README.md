## Widget

### Root and Structural Widgets
- MaterialApp: Widget root yang mengatur aplikasi Material Design.

- Scaffold: Menyediakan struktur halaman dengan AppBar dan body.

- AppBar: Bilah atas halaman dengan judul.

### Layout Widgets
- Column: Mengatur widget child dalam kolom vertikal.

- Container: Wadah dengan margin, padding, warna, dll.

- AspectRatio: Memaksa rasio child menjadi spesifik untuk lebar dan tinggi.

- Center: Memusatkan di tengah.

- Row: Mengatur widget child dalam baris horizontal.
  
### UI Element Widgets
- Image.network: Menampilkan gambar dari URL.

- Text: Menampilkan teks.

- IconButton: Sebuah button clickable yang berupa icon. untuk memanggil _incrementCounter function.

- Icon: Menampilkan ikon.
  
### State Management Widgets
- StatefulWidget: Widget yang memiliki state, seperti CounterCard.

- State: Kelas yang mengelola state untuk StatefulWidget.
